# SMCP — Smart City Complaint Portal

A MERN stack web app for citizens to register and track complaints for smart city services.

## Tech stack
MongoDB · Express.js · React · Node.js

## Getting started
```bash
git clone https://github.com/dev45221904/smcp.git
cd smcp
    // For backend
        cd server
        npm install
        npm run dev
    // for front-end
        npm install
        npm run dev
```

## Features
- Submit complaints with category and location
- Track complaint status in real time
- Admin dashboard to manage and resolve complaints
